# KeepBooking

**KeepBooking** — это современная платформа для онлайн-бронирования столиков в кафе и ресторанах, объединяющая пользователей и заведения в единой экосистеме.

Проект предоставляет возможность быстро находить кафе и рестораны, просматривать информацию о заведениях, выбирать свободные столики на интерактивной схеме зала и оформлять бронирование в несколько кликов. Владельцы заведений получают удобную систему управления ресторанами, залами, столиками и бронированиями через веб-панель администратора.

Основная цель проекта — сделать процесс бронирования максимально простым и доступным как для пользователей, так и для бизнеса. На первом этапе сервис разрабатывается как полностью бесплатная платформа без комиссий и абонентской платы, что позволяет любому заведению бесплатно подключиться к системе.

## Основные возможности

### Для пользователей

* Регистрация и авторизация.
* Управление личным профилем.
* Выбор страны и города проживания.
* Поиск кафе и ресторанов.
* Просмотр заведений на карте.
* Просмотр фотографий, описания и информации о ресторане.
* Фильтрация по кухне, рейтингу, местоположению и другим параметрам.
* Просмотр схемы зала.
* Выбор конкретного столика.
* Онлайн-бронирование.
* История посещений и бронирований.
* Избранные заведения.
* Push-уведомления о статусе бронирования.
* Возможность оставлять отзывы после посещения.

### Для ресторанов

* Управление компанией и несколькими филиалами.
* Добавление и редактирование заведений.
* Настройка графика работы.
* Загрузка фотографий.
* Управление меню.
* Создание интерактивной схемы зала.
* Добавление столиков с указанием количества мест, типа (обычный, диван, VIP и т.д.) и расположения.
* Просмотр и обработка бронирований.
* Подтверждение или отмена бронирования.
* Просмотр аналитики по загрузке заведения.

### Для администратора системы

* Управление пользователями.
* Управление компаниями.
* Модерация заведений.
* Управление справочниками (страны, города, категории кухни).
* Просмотр общей статистики платформы.
* Мониторинг работы системы.

## Архитектура проекта

Проект разрабатывается по принципам современной модульной архитектуры с разделением на независимые функциональные модули.

В состав системы входят:

* Backend (Java, Spring Boot)
* Мобильное приложение для пользователей
* Веб-админка для ресторанов
* Панель администратора платформы
* REST API
* PostgreSQL
* Redis
* Docker
* OpenAPI (Swagger)

Архитектура проектируется с учетом возможности дальнейшего масштабирования и выделения отдельных модулей в микросервисы без существенной переработки системы.

## Цели проекта

* Создать единую платформу для онлайн-бронирования столиков.
* Предоставить ресторанам бесплатный инструмент управления бронированиями.
* Предоставить пользователям удобный сервис поиска и бронирования заведений.
* Реализовать современную архитектуру уровня production.
* Получить практический опыт разработки полного цикла — от проектирования базы данных и REST API до мобильного приложения, веб-интерфейса и развертывания инфраструктуры.

## Перспективы развития

В дальнейшем в платформу могут быть добавлены:

* очередь ожидания при отсутствии свободных столиков;
* предварительный заказ блюд;
* программа лояльности;
* онлайн-оплата депозитов;
* интеграция с POS-системами ресторанов;
* рекомендации на основе предпочтений пользователя;
* расширенная аналитика для бизнеса;
* поддержка нескольких языков и международный запуск.
